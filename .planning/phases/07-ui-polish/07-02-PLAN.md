## 任务清单

### Task 1: Sidebar 响应式收起

修改 `App.vue` Sidebar 组件:
- 窗口宽度 >= 1200px: 正常显示（240px）
- 窗口宽度 < 1200px: Sidebar 收起为图标栏（48px），显示图标按钮
- 收起状态下hover展开完整Sidebar（overlay模式，不挤压主内容）
- 使用 CSS `@media` + Vue `windowWidth` 响应式

```css
/* 图标栏模式 */
.sidebar.collapsed {
  width: 48px;
  overflow: hidden;
}
.sidebar.collapsed:hover {
  width: 240px;
  position: absolute;
  z-index: 100;
  box-shadow: var(--ui-shadow-md);
}
```
