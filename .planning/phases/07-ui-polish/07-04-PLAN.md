## 任务清单

### Task 1: 通用 Toast composable `useToast.ts`

```typescript
// frontend/src/composables/useToast.ts

type ToastType = 'success' | 'error' | 'warning' | 'info'

interface ToastOptions {
  type: ToastType
  title: string
  message?: string
  duration?: number    // 自动消失时间(ms)，0=不自动消失
  action?: {
    label: string
    handler: () => void
  }
}

// 核心:
// - 全局单例 (provide/inject)
// - toasts: Ref<ToastItem[]>
// - show(options) → 添加到队列
// - success(title, message?), error(title, message?), warning(...), info(...)
// - remove(id) → 手动移除
// - 最大同时显示5条，超出自动移除最早的
```

快捷方法:
```typescript
const toast = useToast()
toast.success('保存成功')
toast.error('导出失败', 'PDF导出过程中发生错误')
toast.warning('自动备份未启用', '建议启用自动备份保护数据', { action: { label: '去设置', handler: openSettings } })
```

### Task 7: 确认对话框替代 confirm()

将现有的 `window.confirm()` 调用替换为自定义确认对话框:
- 删除简历
- 永久删除
- 回滚快照
- 恢复备份（覆盖当前数据）

创建 `ConfirmDialog.vue` 通用确认组件:
- 标题 + 消息 + 确认/取消按钮
- 危险操作使用红色确认按钮
- 支持通过 composable 编程式调用

```typescript
const { confirm } = useConfirm()
const ok = await confirm({
  title: '删除简历',
  message: '该简历将移入回收站，30天内可恢复',
  confirmLabel: '删除',
  type: 'danger'
})
```
