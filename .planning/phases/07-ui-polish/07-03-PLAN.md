## 任务清单

### Task 1: 全局快捷键注册器 `useKeyboard.ts`

```typescript
// frontend/src/composables/useKeyboard.ts

interface ShortcutBinding {
  id: string          // 唯一标识，如 'ai.polish'
  label: string       // 显示名称，如 'AI润色'
  defaultKey: string  // 默认键绑定，如 'Ctrl+Shift+R'
  customKey?: string  // 用户自定义覆盖
  handler: () => void // 执行函数
  scope: 'global' | 'editor' | 'chat' // 作用域
  enabled: boolean
}

// 核心功能:
// - 注册快捷键绑定
// - 全局 keydown 监听，匹配时执行handler
// - 支持作用域（编辑器激活时优先editor scope）
// - 防止与Monaco内置快捷键冲突
// - provide/inject 全局单例
```

关键设计:
- 使用 `keydown` 事件监听 `event.ctrlKey/metaKey + event.key`
- Monaco编辑器获得焦点时，跳过Monaco已占用的快捷键
- `scope` 控制不同场景下的快捷键可用性

### Task 2: 定义默认快捷键映射

```typescript
const DEFAULT_SHORTCUTS: ShortcutBinding[] = [
  // AI操作
  { id: 'ai.polish',   label: 'AI润色',   defaultKey: 'Ctrl+Shift+R', scope: 'editor' },
  { id: 'ai.translate', label: 'AI翻译',   defaultKey: 'Ctrl+Shift+T', scope: 'editor' },
  { id: 'ai.shorten',  label: 'AI缩写',   defaultKey: 'Ctrl+Shift+D', scope: 'editor' },

  // 视图
  { id: 'view.togglePreview', label: '切换预览', defaultKey: 'Ctrl+Shift+V', scope: 'global' },
  { id: 'view.toggleChat',    label: 'AI对话',   defaultKey: 'Ctrl+Shift+A', scope: 'global' },

  // 保存
  { id: 'file.save',    label: '保存',      defaultKey: 'Ctrl+S',       scope: 'global' },
]
```

注意: 使用 `Ctrl+Shift+*` 而非 `Ctrl+*`，避免与Monaco内置快捷键冲突。需求文档中的 Ctrl+R/T/D 在实际实现中使用 Shift 修饰符保护。

### Task 4: 快捷键执行逻辑接入

在 `EditorView.vue` 中注册快捷键handler:
- `ai.polish` → 触发选区文本的AI润色操作
- `ai.translate` → 触发AI翻译
- `ai.shorten` → 触发AI缩写
- `view.togglePreview` → 切换预览面板显示
- `view.toggleChat` → 切换AI对话侧边栏
- `file.save` → 手动保存（替换现有的硬编码 keydown 监听）

AI快捷键需要先检查是否有选中文本，无选中文本时不执行或显示提示。

### Task 5: 快捷键配置面板 — 设置对话框"快捷键"Tab

在 07-01 创建的 `SettingsDialog.vue` 中添加"快捷键"Tab:
- 列表显示所有快捷键：名称 + 当前键绑定
- 点击键绑定进入"监听模式"，按键后捕获新键绑定
- 支持恢复默认
- 冲突检测：同一键绑定被多个操作使用时标红警告
- 保存后立即生效

```vue
<!-- 快捷键配置项 -->
<div class="shortcut-item">
  <span class="shortcut-label">{{ binding.label }}</span>
  <kbd class="shortcut-key" @click="startListening(binding.id)">
    {{ formatKey(binding.customKey || binding.defaultKey) }}
  </kbd>
  <button @click="resetShortcut(binding.id)">重置</button>
</div>
```

注意: 此 Tab 的父容器 `SettingsDialog.vue` 已在 07-01 Task 5 创建，此处仅需实现 Tab 内容面板组件 `ShortcutSettingsPanel.vue` 并注册到 SettingsDialog。
