### 3. 自动保存 Composable (`frontend/src/composables/useAutoSave.ts`)

**功能 per D-25~D-28:**
- 从 EditorView 的内联保存逻辑抽取为独立 composable
- 管理保存状态：「已保存」(绿色) / 「保存中...」(灰色) / 「未保存」(橙色) / 「保存失败」(红色)
- 自动保存触发条件：
  - 内容变更后 30 秒间隔
  - AI 操作完成时（外部手动调用 `triggerSave()`）
  - 页面切换/离开时（beforeRouteLeave 守卫）
- 保存失败时显示错误状态 + 重试按钮
- 不阻塞用户继续编辑

```typescript
export function useAutoSave(options: {
  resumeId: Ref<string>
  getData: () => { markdownContent: string; jobTarget: string }
}) {
  const saveStatus: Ref<'saved' | 'saving' | 'unsaved' | 'error'> = ref('saved')
  const errorMessage: Ref<string> = ref('')

  let autoSaveTimer: ReturnType<typeof setInterval> | null = null
  let isDirty = false

  // 标记内容已修改
  function markDirty(): void

  // 手动触发保存
  async function triggerSave(): Promise<void>

  // 启动 30 秒自动保存
  function startAutoSave(): void

  // 停止自动保存（离开页面时调用）
  function stopAutoSave(): void

  return { saveStatus, errorMessage, markDirty, triggerSave, startAutoSave, stopAutoSave }
}
```

**保存逻辑：**
```typescript
async function save() {
  saveStatus.value = 'saving'
  try {
    const data = options.getData()
    await UpdateResume(options.resumeId.value, JSON.stringify(data))
    saveStatus.value = 'saved'
    isDirty = false
    errorMessage.value = ''
  } catch (err) {
    saveStatus.value = 'error'
    errorMessage.value = String(err)
  }
}

// 30 秒自动保存
function startAutoSave() {
  stopAutoSave()
  autoSaveTimer = setInterval(() => {
    if (isDirty) save()
  }, 30_000)
}
```

### 二次确认弹窗

永久删除使用浏览器原生 `confirm()`，与 Phase 3 的错误处理风格一致：
```typescript
async function permanentDelete(id: string) {
  if (!confirm('永久删除后无法恢复，确定要删除吗？')) return
  await PermanentDeleteResume(id)
  await fetchDeletedResumes()
}
```

### 自动保存时序

```
用户输入 → handleContentChange() → content ref 更新
                                    ↓
                              150ms debounce → debouncedContent 更新（触发预览渲染）
                                    ↓
                              markDirty() → isDirty = true
                                    ↓
                              30s timer → if isDirty → triggerSave()
```

注意：预览防抖 (150ms) 和自动保存 (30s) 是两个独立机制，不互相干扰。
