# Plan 03-02: 选区浮动工具栏

**Phase:** 3 of 7 — AI 核心能力
**Status:** Planned
**Depends on:** 03-01 (SSE infrastructure)

## Goal

用户框选编辑器文本后，显示浮动工具栏，提供 AI 润色/翻译/缩写/重写四个快捷操作。

## Requirements Covered
- **EDIT-07**: 选区快捷操作 — 浮动工具栏，包含 AI 润色/翻译/缩写/重写

## Deliverables

### 1. AIFloatingToolbar 组件 (`frontend/src/components/AIFloatingToolbar.vue`)

**组件 props：**
```typescript
interface Props {
  visible: boolean
  position: { top: number; left: number }
  selectedText: string
  operationId: string
}
```

**UI 设计：**
- 4 个操作按钮横向排列：润色 | 翻译 | 缩写 | 重写
- 每个按钮带图标 + 文字标签
- 加载状态：按钮内显示 spinner
- 最小高度 36px，圆角 6px，VS Code 暗黑风格背景

**交互流程：**
1. 用户选中文字 → 计算位置 → 显示工具栏
2. 用户点击操作 → 显示 loading → 调用 `useAIStream`
3. 流式返回 → 替换选区文字
4. 操作完成 → 隐藏工具栏

**位置计算：**
```typescript
// 使用 Monaco API
const pos = editor.getScrolledVisiblePosition(selection.getStartPosition())
// 转换为 DOM 坐标，设置 top/left
// 超出视口时自动翻转（向上显示）
```

**边界处理：**
- 选区为空：不显示
- 选区跨多行：显示在选区起始位置上方
- 工具栏超出右边界：左偏移对齐

### 2. useAISelection Composable (`frontend/src/composables/useAISelection.ts`)

**核心功能：**
- 监听 Monaco `onDidChangeCursorSelection` 事件
- 检测是否有有效选区
- 计算工具栏位置
- 提供 `performAIOperation(operation: string)` 方法

**操作映射：**
```typescript
const operations = {
  polish:   '润色',
  translate:'翻译',
  summarize:'缩写',
  rewrite:  '重写'
}
```

每个操作的 prompt：
```
润色: "请润色以下简历内容，使语言更专业、精炼：\n{selectedText}"
翻译: "请将以下内容翻译成英文，保持专业语气：\n{selectedText}"
缩写: "请将以下内容压缩到原长度的60%，保留关键信息：\n{selectedText}"
重写: "请重写以下简历内容，换一种表达方式：\n{selectedText}"
```

### 3. MonacoEditor 集成 (`frontend/src/components/MonacoEditor.vue`)

**修改：**
- 导入并渲染 `AIFloatingToolbar` 组件
- 注入 `useAISelection` composable
- 选区变化时更新 toolbar 状态
- 操作完成后调用 `editor.executeEdits` 替换选区

```typescript
// 替换逻辑
editor.executeEdits('ai-operation', [{
  range: selection,
  text: streamedContent,
  forceMoveMarkers: true
}])
```

### 4. 职位目标上下文

**JobTargetChip 组件：** (`frontend/src/components/JobTargetChip.vue`)

## Technical Details

### 工具栏渲染策略
- 使用 Vue `<Teleport to="body">` 避免 z-index 问题
- `position: fixed` 定位
- 工具栏本身不在 Monaco editor DOM 内

```typescript
})
```
