# Plan 03-01: Claude API 适配与 SSE 流式传输

**Phase:** 3 of 7 — AI 核心能力
**Status:** Planned
**Depends on:** Phase 2 complete

## Goal

建立 AI 能力基础设施：Go 后端调用 Claude Messages API，实时流式传输响应到前端，打字机效果渲染。

## Requirements Covered
- **AIAI-01**: Claude Messages API 核心适配，支持自定义 BaseURL
- **AIAI-03**: 默认模型、最大 Token 数、超时时间配置
- **AIAI-07**: 流式传输（SSE 协议），打字机效果

## Deliverables

### 1. Go 后端 AI 客户端 (`backend/internal/ai/client.go`)

**核心功能：**
- `AIClient` struct：封装 HTTP 客户端，支持自定义 BaseURL 和 API Key
- `Chat(ctx, model, messages, maxTokens, timeout)` 方法：发送非流式请求
- `ChatStream(ctx, model, messages, maxTokens, timeout)` 方法：返回 `io.ReadCloser`（SSE 流）
- `ValidateAPIKey(apiKey, baseURL)` 方法：发送 test 请求验证 Key 有效性
- `BuildMessages(systemPrompt, userContent, jobTarget, includeFullContext)` 工具函数

**API 端点：**
- POST `{baseURL}/v1/messages`
- Headers: `x-api-key`, `anthropic-version: 2023-06-01`, `content-type: application/json`
- Body: `{ model, max_tokens, stream: true|false, messages: [...] }`

**错误处理：**
- 网络超时 → `AIErrNetwork`
- 401/403 → `AIErrAuth`
- 429 → `AIErrRateLimit`
- 其他 HTTP 错误 → `AIErrAPI`

### 4. 前端 SSE 流处理 (`frontend/src/composables/useAIStream.ts`)

**核心功能：**
- `useAIStream(operationId: string)` composable
- 返回 `{ content, isStreaming, error, abort }`
- 使用 Wails `EventsOn` 监听 `ai:stream:{operationId}` 事件
- 16ms debounce 批量更新 DOM
- `abort()` 调用 `EventsOff` 取消监听

**类型定义 (`frontend/src/types/ai.ts`)：**
```typescript
export interface AIStreamChunk {
  type: 'content' | 'error' | 'done'
  content?: string
  error?: string
}

export interface AIConfig {
  apiKey: string
  baseURL: string
  defaultModel: string
  maxTokens: number
  timeoutSeconds: number
}
```

## Technical Details

### SSE 事件协议

**后端 → 前端事件：**
```go
// 每个 chunk 发送一次
wails.EventsEmit(ctx, "ai:stream:"+operationId, map[string]any{
    "type":    "content",
    "content": chunkText,
})
// 完成时
wails.EventsEmit(ctx, "ai:stream:"+operationId, map[string]any{
    "type": "done",
})
// 错误时
wails.EventsEmit(ctx, "ai:stream:"+operationId, map[string]any{
    "type":  "error",
    "error": errMessage,
})
```

**系统 Prompt 模板：**
