# Plan 03-04: AI 对话侧边栏

**Phase:** 3 of 7 — AI 核心能力
**Status:** Planned
**Depends on:** 03-01 (SSE infrastructure)

## Goal

在编辑器右侧提供 AI 对话侧边栏，支持多轮对话，用户可以引用选中文本、获取简历建议，一键将 AI 输出插入编辑区。

## Requirements Covered
- **AIAI-08**: AI 对话侧边栏 — 编辑器右侧唤起，支持多轮对话，可引用选中文本，输出可一键插入编辑区

### 2. 消息数据模型 (`frontend/src/types/ai.ts`)

```typescript
export interface ChatMessage {
  id: string
  role: 'user' | 'assistant'
  content: string
  timestamp: number
  quotedText?: string  // 引用的选中文本
}

export interface ChatConversation {
  id: string
  resumeId: string
  messages: ChatMessage[]
  createdAt: number
}
```

### 3. 后端对话管理

**Bridge 方法：**
```go
func (a *App) GetChatHistory(resumeId string) ([]ChatMessage, error)
func (a *App) SaveChatMessage(resumeId string, message ChatMessage) error
func (a *App) ClearChatHistory(resumeId string) error
```

**数据库 Schema：**
```sql
CREATE TABLE IF NOT EXISTS ai_messages (
    id TEXT PRIMARY KEY,
    resume_id TEXT NOT NULL,
    role TEXT NOT NULL CHECK(role IN ('user','assistant')),
    content TEXT NOT NULL,
    quoted_text TEXT,
    created_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (resume_id) REFERENCES resumes(id) ON DELETE CASCADE
);
```

### 4. 引用选中文本功能

**触发：** 侧边栏打开时，如果编辑器有选中文本，显示引用按钮

**逻辑：**
1. 点击「📎 引用」→ 获取当前选中文本
2. 用户输入消息 → 消息追加 `"引用：{quotedText}\n\n{userMessage}"` 格式发送

### 5. 插入编辑区功能

**「📋 插入」按钮：** 出现在每条 AI 回复下方

**逻辑：**
1. 点击「插入」→ 获取 AI 回复内容
2. 调用 `editor.executeEdits` 在当前光标位置插入
3. 插入完成后显示 toast 确认

### 6. Sidebar 切换

**触发方式：**
- 编辑器工具栏按钮
- 快捷键 `Ctrl/Cmd + Shift + A`（Phase 7 实现快捷键体系时）

**动画：** 300ms ease-out 滑入/滑出

### 7. 对话上下文注入

**系统 Prompt（每条消息追加）：**
```
当前简历编辑上下文：
- 目标岗位：{jobTarget}
- 当前简历内容：{markdownContent}（当 includeFullContext=true 时）

请以简历优化助手的身份回答。
```

**历史消息传递：** 最近 10 条消息作为上下文（token 预算控制）

## Technical Details

### 流式渲染
```typescript
// 每收到一个 chunk，更新对应消息的 content
// 使用 requestAnimationFrame 节流 DOM 更新
const rafId = ref<number>()
watch(streamingContent, (newContent) => {
  cancelAnimationFrame(rafId.value)
  rafId.value = requestAnimationFrame(() => {
    messages.value[messages.value.length - 1].content = newContent
  })
})
```

### 滚动自动跟进
```typescript
watch(() => messages.value.length, () => {
  nextTick(() => {
    scrollContainer.value?.scrollTo({
      top: scrollContainer.value.scrollHeight,
      behavior: 'smooth'
    })
  })
})
```

### 对话持久化
- 每次用户发送后立即保存到 SQLite
- AI 回复流式完成后保存（包含完整内容）
- 页面刷新后对话历史完整保留
