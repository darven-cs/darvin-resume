# Plan 03-06: AI 异常兜底处理

**Phase:** 3 of 7 — AI 核心能力
**Status:** Planned
**Depends on:** 03-01 (SSE infrastructure)

## Goal

覆盖 AI 调用的所有异常场景：网络失败、Token 超限、格式异常、用户中断，提供清晰的错误提示和可执行的恢复方案，确保无数据丢失。

## Requirements Covered
- **AIAI-10**: 网络/API 调用失败时弹窗提示错误原因，完整保留原始输入，提供一键重试
- **AIAI-11**: Token 超限时自动检测并分批生成，同时弹窗提示用户精简内容
- **AIAI-12**: AI 返回格式异常时先基于 Schema 校验，不通过自动重试 1 次，仍失败则完整返回原始内容
- **AIAI-13**: 用户主动中断生成时保留已生成的有效内容，允许继续编辑或重新生成

#### 4.3 格式异常 (AIAI-12)
```go
// 后端自动重试逻辑
func (c *AIClient) ChatWithRetry(ctx, ..., maxRetries int) (string, error) {
    for i := 0; i < maxRetries; i++ {
        response, err := c.Chat(ctx, ...)
        if err != nil {
            return "", err
        }
        if !isValidJSON(response) {
            // 重试一次
            continue
        }
        return response, nil
    }
    // 两次都失败，返回原始响应
    return response, AIErr{
        Code: "FORMAT_ERROR",
        Message: "AI 返回格式异常，已保留原始输出",
        Recoverable: true,
    }
}
```

#### 4.4 用户中断 (AIAI-13)
```typescript
// useAIStream.ts
function abort() {
  // 1. 通知后端取消操作
  backend.AICancelOperation(operationId)
  // 2. 停止监听事件
  wails.EventsOff(`ai:stream:${operationId}`)

```

**后端取消：**
```go
// 维护活跃操作 map
var activeOperations = sync.Map{}

func (a *App) AISendMessage(operationId string, ...) {
    activeOperations.Store(operationId, true)
    defer activeOperations.Delete(operationId)

    // 检查是否已取消
    if _, ok := activeOperations.Load(operationId); !ok {
        return "", errors.New("operation cancelled")
    }
}

func (a *App) AICancelOperation(operationId string) error {
    activeOperations.Delete(operationId)
    return nil
}
```

## Technical Details

### 错误恢复策略
| 错误类型 | 用户体验 | 恢复方案 |
|---------|---------|---------|
| NETWORK_ERROR | Toast + 重试按钮 | 一键重试 |
| TIMEOUT | Toast + 重试按钮 | 增加超时重试 |
| AUTH_FAILED | Modal 警告 | 引导去设置页 |
| RATE_LIMIT | Toast + 倒计时 | 等待后重试 |
| TOKEN_LIMIT | Modal 详情 | 引导精简内容 |
| FORMAT_ERROR | Toast + 保留原始 | 允许手动编辑 |
| ABORTED | Toast 轻提示 | 保留已生成内容 |

### 数据不丢失保证
- 所有原始输入在操作开始时缓存
- 流式内容实时累积，不因错误丢失
- 操作失败后原文不变，不覆盖用户内容
- 重试使用缓存的原始参数
