# Plan 03-03: Diff 对比视图

**Phase:** 3 of 7 — AI 核心能力
**Status:** Planned
**Depends on:** 03-01 (AI stream) + 03-02 (floating toolbar)

## Goal

AI 修改完成后，以 diff 视图展示原文与修改后的差异，用户可以一键接受或拒绝修改。

## Requirements Covered
- **EDIT-11**: Diff 对比视图 — AI 修改内容后以 Diff 视图展示差异（删除标红/新增标绿），用户点击接受/拒绝

## Deliverables

### 1. AIDiffView 组件 (`frontend/src/components/AIDiffView.vue`)

**显示位置：** 编辑器下方（选区下方），作为内联区块插入，非模态框，不遮挡预览区

**UI 设计：**
```
┌─────────────────────────────────────────────────┐
│  AI 修改对比                      [接受] [拒绝]   │
├─────────────────────────────────────────────────┤
│  - 原文：删除了这个冗余的描述，使语言更精炼       │  ← 红色背景 #ffcccc
│  + 新文：优化了描述，突出技术深度                 │  ← 绿色背景 #ccffcc
│  - 删除了重复的项目描述                          │
│  + 整合了项目亮点，突出了技术挑战                │
└─────────────────────────────────────────────────┘
```

**组件 props：**
```typescript
interface Props {
  originalText: string
  modifiedText: string
  operationType: 'polish' | 'translate' | 'summarize' | 'rewrite'
}
```

**组件 events：**
- `@accept`: 用户点击接受
- `@reject`: 用户点击拒绝

**样式：**
- 圆角边框，左侧 3px 颜色条（蓝色 #409eff）
- 删除行：红色背景 `rgba(255,0,0,0.08)`
- 插入行：绿色背景 `rgba(0,255,0,0.08)`
- 等号开头的相同行：灰色淡化显示
- 字体：13px，等宽字体

### 2. Diff 计算逻辑

**使用 `diff` npm 包（`diff@^5.2.0`）：**
```typescript
import { diffLines } from 'diff'

const diffs = diffLines(originalText, modifiedText)
// 返回: Array<{ value: string, added?: boolean, removed?: boolean }>
```

**转换渲染：**
```typescript
// 过滤掉 removed-only 行的结尾换行修正
diffs.forEach(d => {
  if (d.removed) {
    // 标记为删除行，不显示加号
  } else if (d.added) {
    // 标记为新增行，不显示减号
  }
})
```

### 3. EditorView 集成

**状态管理：**
```typescript
interface AIDiffState {
  visible: boolean
  originalText: string
  modifiedText: string
  operationType: string
  range: monaco.IRange // 用于 accept 时定位替换
  operationId: string
}

const aiDiff = reactive<AIDiffState>({
  visible: false,
  originalText: '',
  modifiedText: '',
  operationType: '',
  range: null,
  operationId: ''
})
```

**流程：**
1. AI 操作开始 → 保存原始选区 `range` 和 `originalText`
2. 流式返回时，累积 `modifiedText`，DiffView 显示对比
3. **接受** → `editor.executeEdits` 替换，DiffView 关闭
4. **拒绝** → DiffView 关闭，原文保持不变

### 4. diff 依赖安装

```bash
cd frontend && npm install diff@^5.2.0 && npm install -D @types/diff@^5.2.0
```

## Technical Details

### Diff 位置策略
- DiffView 作为 `AIFloatingToolbar` 的兄弟元素，叠放在一起
- 工具栏在 diff 上方，diff 在工具栏下方
- 位置随工具栏位置联动

### 性能优化
- diff 计算在流式返回完成后再执行（避免每次 chunk 都重算）
- 对于超长文本（>10000字），diff 计算放入 Web Worker

### 撤销栈集成
```typescript
// 接受修改时，合并到单个 undo 单元
editor.executeEdits('ai-accept', [{
  range: aiDiff.range,
  text: aiDiff.modifiedText,
  forceMoveMarkers: true
}])
// 之后 Ctrl+Z 一次性撤销整个 AI 修改
```
