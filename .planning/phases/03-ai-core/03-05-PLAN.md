**Phase:** 3 of 7 — AI 核心能力

- **AIAI-09**: 一键解析 — 用户粘贴纯文本/Markdown 旧简历后 AI 自动解析生成结构化数据+Markdown 内容

## Technical Details

### 解析流程
```
用户粘贴 → 验证非空 → 发送解析请求 → 流式显示"正在解析..."
     → 解析完成 → 验证 JSON schema → 显示预览
     → 用户确认 → JSON → Markdown → 填充编辑器
```

### JSON Schema 验证
```typescript
// 使用简单验证函数（Phase 1 已定义）
function validateParsedResume(data: unknown): data is Resume {
  const r = data as Resume
  return (
    typeof r.name === 'string' &&
    Array.isArray(r.education) &&
    Array.isArray(r.experience)
  )
}
```

### 异常处理
- AI 返回非 JSON 格式 → 提示用户并允许手动编辑
- Schema 验证失败 → 提示具体缺失字段，允许跳过该字段
- 解析超时 → 重试或手动编辑
